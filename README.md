# Wheeled-Biped

## Wheel profile generator

The wheel profile generator lives in [WheelProfile/GenerateSVG.py](WheelProfile/GenerateSVG.py).

Generate the default SVG:

```powershell
python WheelProfile\GenerateSVG.py
```

Launch the Dear PyGui editor:

```powershell
python WheelProfile\GenerateSVG.py --gui
```

If Dear PyGui is not installed yet:

```powershell
pip install dearpygui
```
